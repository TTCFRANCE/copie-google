# Google Réplique (HTML / CSS)

## Description
Ce projet est une **réplique front-end de la page d'accueil de Google**.  
L'objectif était de recréer visuellement l'interface principale de Google avec **HTML et CSS**, en incluant :  
- La barre de navigation supérieure (Gmail, Images, etc.)  
- Le logo Google en SVG  
- La barre de recherche et les boutons "Google Search" et "I'm Feeling Lucky"  
- Le footer avec les liens Google

Ce projet ne contient pas la logique de recherche réelle, il s'agit uniquement d'une **reproduction visuelle interactive**.

## Objectif
- Améliorer mes compétences en **HTML et CSS**
- Reproduire fidèlement une interface web célèbre
- Expérimenter le positionnement, les flexbox, les icônes SVG et le responsive design

## Fonctionnalités
- Navbar fonctionnelle avec liens (ouverts dans le navigateur)
- Barre de recherche avec placeholder
- Logo Google en SVG responsive
- Footer fixe en bas de page
- Effets hover sur les liens et les icônes

## Technologies utilisées
- HTML pour la structure de la page
- CSS pour le style, le positionnement et les effets visuels
- SVG pour le logo Google

## Lancer le projet
1. Ouvrir le fichier `index.html` dans un navigateur web  
2. Naviguer sur la page comme sur Google  
3. Tester les liens et la barre de recherche (texte seulement, pas de moteur)

## Améliorations possibles
